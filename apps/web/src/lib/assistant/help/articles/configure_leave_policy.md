Leave policies define how many days each leave type gives employees per year, and whether unused days carry over. Each new employee inherits the current policies automatically.

1. Open **Settings** from the left sidebar.
2. Scroll to the **Leave Policies** section.
3. Click **Add Policy** to create a new leave type, or click the edit icon on an existing one.
4. Enter the **Leave Type** name, **Days Per Year**, and whether **Carry Forward** is allowed.
5. If carry-forward is on, enter the **Maximum Carry-Forward Days**.
6. Click **Save**.

Changes apply to all employees' balances going forward. Existing approved leaves for the current year are not retroactively adjusted.
