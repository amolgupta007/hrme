1. Go to **JambaGeo → Geofences**.
2. Click **New geofence**.
3. Enter a **name** (e.g. "Head Office" or "Client A Site").
4. On the map, click to set the **centre point**, then drag the radius handle to set the zone size.
5. Optionally set an **address label** and **notes**.
6. Click **Save geofence**.

The geofence appears as a circle on the map. In a later phase, the mobile app will use these zones to auto-trigger check-in/check-out for field staff. You can edit or delete a geofence by clicking it on the map.
