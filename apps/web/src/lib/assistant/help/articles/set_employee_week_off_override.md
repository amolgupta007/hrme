# Override one employee's week-off schedule

Use this when one employee follows a different week-off schedule from the rest of
the org — e.g. a 6-day worker in a 5-day org, or a manager who works Saturdays.

## Steps

1. Open **Settings → Attendance → Per-employee week-off overrides**.
2. Click **Add override**.
3. Pick the **employee**.
4. Choose **5-day** (2 off days) or **6-day** (1 off day).
5. Tick the day chip(s) to mark as off.
6. Optionally set the **Alternate Saturday rule** (None / 1st+3rd / 2nd+4th).
7. Set the **Effective from** date.
8. Click **Save override**.

To revert an employee to the org policy, click the trash icon next to their row.
