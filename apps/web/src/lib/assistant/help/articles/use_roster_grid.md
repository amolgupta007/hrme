# Use the roster grid

## Steps

1. Open **Attendance → Roster**.
2. The grid shows your team (employees as rows) for the current week (Mon–Sun).
3. Drag a shift chip from the left palette into any employee × day cell.
4. The assignment saves immediately. The cell shows the shift name.
5. Click a "tentative" cell (rotational, lighter chip with `?`) to **promote it to fixed**.
6. Use **prev/next week** arrows in the header to navigate.

**Manager scope:** if you're a manager, you only see employees in departments where you're the head.

**Soft warnings:** the system warns you (but doesn't block) if you're assigning over an existing assignment, a week-off, or using an inactive shift.
