Admins can add employees one at a time from the Employees page. Once saved, you can optionally send a Clerk sign-up invite so the employee can log in.

1. Open **Employees** from the left sidebar.
2. Click the **Add Employee** button in the top-right corner.
3. Fill in the employee's first name, last name, email, department, role, and employment type.
4. Set the **Status** to Active (default) unless this is a future-dated hire.
5. Toggle **Send invite email** on if you want Clerk to email them a sign-up link right away.
6. Click **Save** to create the employee record.

The new employee appears in the directory immediately. If you sent an invite, they'll receive a sign-up email within a minute. You can always send an invite later from the employee's action menu.
