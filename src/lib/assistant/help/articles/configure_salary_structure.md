A salary structure defines how an employee's CTC is broken down into components like Basic, HRA, PF, and TDS. You need to set this up before running payroll for each employee.

1. Open **Payroll** from the left sidebar.
2. Click the **Salary Structures** tab.
3. Find the employee and click **Configure** (or **Edit** if a structure already exists).
4. Enter the **Annual CTC** in rupees.
5. Choose the employee's **State** and toggle **Metro City** on or off (affects HRA calculation).
6. Select the **Tax Regime** — New (default) or Old. If Old, enter the total **Additional Deductions** (80C + 80D + 24B combined annual amount).
7. Click **Save Structure**.

JambaHR auto-computes Basic, HRA, Special Allowance, PF, Professional Tax, and projected TDS. The employee can see their breakdown in the **My Compensation** tab. One structure is saved per employee — saving again updates the existing one.
